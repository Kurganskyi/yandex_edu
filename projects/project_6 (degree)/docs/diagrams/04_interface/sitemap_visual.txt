BOTTOM NAVIGATION (Основная навигация):
═══════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│  📊 Dashboard  │  🤖 Scenarios  │  ⚙️  Settings         │

• Dashboard - главный экран управления домом
• Scenarios - создание и управление сценариями автоматизации
• Settings - настройки профиля, дома и приложения

                                                    СОСТОЯНИЯ ЭКРАНОВ
════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

ПУСТЫЕ СОСТОЯНИЯ:
═════════════════

• Пустой дом       - приветствие + "Добавить первое устройство"
• Пустая комната   - иллюстрация + "Добавить устройство в комнату"
• Нет сценариев    - "Создать первый сценарий"

СОСТОЯНИЯ ЗАГРУЗКИ:
═══════════════════

• Skeleton loaders     - для списков устройств/комнат
• Progress indicators  - для длительных операций
• Pull-to-refresh      - для обновления данных

СОСТОЯНИЯ ОШИБОК:
══════════════════

• Network errors        - "Попробовать снова"
• Device unavailable    - индикатор + "Переподключить"
• Validation errors     - inline сообщения
